# <img src="src/logo.ico" width="30"> Hexium

> Launcher de Minecraft minimalista, portátil e leve.

O **Hexium** é um launcher de Minecraft minimalista, focado em desempenho e portabilidade.  
Feito para quem quer jogar rápido, sem complicação e sem depender de launchers pesados.

---

## 🚀 Destaques

- ⚡ **Ultra leve** – pesa menos de 40mb
- 📦 **Portátil** – roda sem instalação  
- 🚀 **Rápido** – abre e inicia o jogo em poucos segundos  
- 🎯 **Minimalista** – interface simples e direta  
- 🧩 **Suporte a skins personalizadas**  

---

## 🖥️ Compatibilidade

- Windows
- Linux
- MacOS (em Breve!)

---

## 🛠️ Tecnologias

- 100% Python 🐍
